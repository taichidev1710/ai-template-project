# Glossary (thuật ngữ dùng chung)

Bộ từ vựng dùng chung để mọi repo gọi cùng một khái niệm theo cùng một tên.

- **User** — một tài khoản có thể đăng nhập (xem data-model).
- **Order** — một đơn mua hàng do một User tạo.
- **Role** — cấp độ truy cập: `admin | editor | viewer` (RBAC).
- **RBAC** — kiểm soát truy cập theo vai trò (role-based access control); được thực
  thi ở phía server.
- **Contract** — thỏa thuận API giữa frontend và backend.
- **Standards** — các quy tắc cấp platform được phân phối từ repo này.
- **Consumer repo** — bất kỳ repo nào sync standards về (frontend, backend, ...).
